# Source Citations

These citations document the intended empirical sources for a conference-grade
FinTTA build. The committed `data/sample` files do not contain licensed data.

- CRSP U.S. Stock Databases: https://www.crsp.org/research/crsp-us-stock-databases/
- CRSP/Compustat Merged Database: https://www.crsp.org/research/crsp-compustat-merged-database/
- WRDS CRSP/Compustat linking overview: https://wrds-www.wharton.upenn.edu/pages/wrds-research/database-linking-matrix/linking-crsp-with-compustat/
- ALFRED archival/vintage macro data: https://alfred.stlouisfed.org/
- FRED vintage-date API documentation: https://fred.stlouisfed.org/docs/api/fred/series_vintagedates.html
- Kenneth R. French Data Library: https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/data_library.html
- SEC EDGAR API documentation: https://www.sec.gov/edgar/sec-api-documentation
- FINRA Short Sale Volume catalog: https://www.finra.org/finra-data/browse-catalog/short-sale-volume
- NYSE historical TAQ trades: https://www.nyse.com/market-data/historical/taq-trades
- FRED VIXCLS series: https://fred.stlouisfed.org/series/VIXCLS
