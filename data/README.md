# FinTTA Data Layer

This directory is structured for the empirical panel used by:

`FinTTA: Fully Test-Time Adaptation for Financial Tabular Prediction`

The conference-grade dataset requires licensed, point-in-time financial data.
No CRSP, Compustat, TAQ, GICS, ETF holdings, or vendor event data is committed
to this repository. If those inputs are unavailable, the scripts build only the
tiny schema-compatible sample under `data/sample`.

## Directory Contract

- `raw/`: uncommitted vendor exports, for example CRSP daily stock files,
  CRSP delisting returns, CRSP/Compustat Merged fundamentals, ALFRED vintages,
  TAQ/liquidity exports, GICS or equivalent sector history, ETF holdings, event
  feeds, and short-interest data.
- `intermediate/`: generated normalized tables such as `prices_daily.parquet`,
  `universe_daily.parquet`, `macro_asof.parquet`, `graph_edges_daily.parquet`,
  and `market_state_daily.parquet`.
- `processed/`: generated full empirical outputs. These are not committed
  unless your license and storage policy allow it.
- `sample/`: committed tiny sample files for tests and demos only.
- `external_docs/`: source citation notes and vendor documentation pointers.

## Source Citations

Primary intended sources are CRSP U.S. Stock Databases for active and inactive
daily equities, CRSP delisting returns, exchange/share codes, and corporate
actions; CRSP/Compustat Merged for PERMNO-GVKEY links and Compustat
fundamentals; ALFRED/FRED for vintage macro; Kenneth French Data Library for
factor returns; SEC EDGAR APIs for filings; FINRA for short-sale volume/short
interest data; and NYSE TAQ or equivalent licensed feeds for microstructure.
See `data/external_docs/source_citations.md` for URLs.

## Leakage Rules

- Features may use only information available on or before `date`.
- Macroeconomic data subject to revision must use vintage/as-of values and keep
  `vintage_date`.
- Fundamentals use filing/public availability dates, not fiscal period end.
- Ticker changes must not create fake new assets; prefer CRSP `PERMNO` or FIGI
  for `asset_id`.
- Do not backfill today’s S&P 500 or Russell 1000 membership into history.
- `label_*`, `forward_return_*`, and `future_norm_return_*` columns are
  evaluation-only and must never enter the adaptation feature list.

## Survivorship-Bias Handling

Conference-grade builds must include active and inactive securities when the
source supports them, retain delisted names on historical dates, and include
delisting returns in `ret_1d`. Current-only ticker universes are prototype-only
and must not be reported as conference-grade evidence.

## Label Generation

Labels are created for 1, 5, and 20 trading-day horizons. For horizon `h`:

`future_norm_return_h = forward_return_h / realized_vol_20_at_t`

Quantile labels use daily cross-sectional buckets:

- `< q10`: `0`, strong sell
- `q10-q35`: `1`, sell
- `q35-q65`: `2`, hold
- `q65-q90`: `3`, buy
- `>= q90`: `4`, strong buy

Absolute labels use thresholds on normalized future return:
`[-1.0, -0.25, 0.25, 1.0]`.

## Open-Data Prototype Now Present

This workspace now also contains a real no-key open-data prototype for
`2014-01-01` through `2024-12-31`. It is intentionally marked
`open-data-prototype / survivor-bias-reduced`, not CRSP-grade.

Downloaded/created files include:

- `data/raw/open_universe_daily.parquet`: PIT S&P 500 business-day universe from
  `fja05680/sp500`, expanded to daily rows.
- `data/raw/open_prices_daily.parquet`: yfinance daily OHLCV/actions for Yahoo-
  available PIT S&P tickers.
- `data/raw/fred_macro_asof_or_latest.parquet`: FRED latest/revised macro,
  rates, credit, VIX, oil, and USD series.
- `data/raw/kenneth_french_daily_factors.parquet`: Kenneth French daily FF5 and
  momentum factors.
- `data/raw/finra_cnms_short_volume.parquet`: FINRA CNMS daily short-sale volume
  proxy from 2018-08-01 onward.
- `data/raw/open/sec_company_tickers.json`: SEC ticker-to-CIK map.
- `data/processed/panel_daily_2015_2024.parquet`: processed open prototype
  panel with labels and causal features.
- `data/processed/source_train_2015_2019.parquet`: source split.
- `data/processed/tta_test_2020_2024.parquet`: TTA split.

The full open-data manifest is:

- `data/raw/open/open_data_manifest.json`
- `data/processed/open_data_summary.json`

Important caveats:

- Yahoo/yfinance does not provide complete CRSP-like delisted history. Missing
  or empty tickers are recorded in `data/raw/open/yfinance_failed_tickers.json`.
- FRED values are latest/revised, not ALFRED vintage. This is acceptable for an
  open prototype, not for the strictest macro-vintage experiment.
- Public sector/industry metadata is current or partially current, not true
  point-in-time GICS.
- FINRA CNMS daily short-sale volume starts in 2018 and is not borrow cost.
- Full SEC 13F/N-PORT holdings are public but multi-GB for this period; they are
  not bulk-downloaded by default into this repo.

## Dataset Grade

The committed sample dataset is still `sample-only`. The new open-data panel is
usable as a free prototype benchmark, but it should be described as
survivor-bias-reduced rather than survivor-bias-free. Conference-grade claims
still require licensed PIT data such as CRSP/Compustat/TAQ or equivalent.
